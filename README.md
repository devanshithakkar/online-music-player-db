# Online Music Player Database (MySQL)

## Description
This project is a DBMS-based Online Music Player system developed using MySQL.
It manages users, artists, albums, songs, playlists, and listening history.

## Features
- User-created playlists (public/private)
- Song, album, and artist management
- Listening history tracking
- Recommendation using subqueries

## Tables
- users
- artists
- albums
- songs
- playlists
- playlist_song
- listening_history
